Core Features

1. System Monitoring (every 5 minutes)

- CPU — Usage %, temperature, clock speed, core count

- RAM — Total, used, available, usage %

- Disk — Space used/free per drive, SSD/HDD type, SMART health

- Battery — Charge level, charging status, wear level, runtime

- Network — Per-adapter: IP, MAC, speed, data sent/received

- Processes — Top 100 running processes with RAM usage

2. Inventory Tracking (every 24 hours)

- Hardware — Manufacturer, model, serial number, BIOS, processor, RAM

- Software — All installed apps with version, publisher, install date

3. Security Monitoring

- Antivirus — Is it running? Signatures up-to-date?

- Firewall — Is it enabled on Domain/Private/Public profiles?

- Login Activity — Who logged in? Failed attempts (possible hacking)?

- USB Activity — When USB drives were plugged in/removed

4. Windows Health

- Services — Are critical Windows services running?

- Updates — How many updates are pending?

- Event Logs — Errors and warnings from System/Application/Security logs

- Startup Programs — What launches automatically at boot

5. Communication

- HTTPS — All data encrypted in transit

- Bearer Token — API key authentication

- Retry (3 attempts) — 10s → 20s → 40s exponential backoff

- Offline Queue — Saves up to 1,000 payloads (100 MB) when server is down

- Auto-flush — Queued data sent automatically when server comes back

6. Reliability

- Never crashes — All errors caught, collector failure doesn't stop others

- Runs as Windows Service — Starts at boot, runs without anyone logged in

- Graceful fallbacks — If WMI fails, try Registry. If COM fails, try WMI. If admin needed, skip gracefully.

- Atomic file writes — No data corruption even on power loss

7. Configuration

- Feature toggles — Enable/disable any collector individually

- Configurable intervals — Collection frequency (default 5 min), inventory (24h)

- No hardcoded values — All settings in appsettings.json

8. Logging

- Console output — See activity in real-time

- File logs — Daily rotating, kept for 30 days

- 4 levels — Info, Warning, Error, Debug

9. Security

- Encryption helper — AES-256 for sensitive config data

- Hardware fingerprint — Unique machine ID from motherboard/CPU/disk

- No secrets in code — API key comes from config, not source

10. Deployment

- Run standalone — dotnet run for testing

- Install as Service — sc create for production

- Self-contained publish — No .NET runtime needed on target machines
